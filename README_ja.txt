
# ========================================================================================
#                                                                                 Overview
#                                                                                 ========
The test project for Go Lang and DBFlute.

とりあえず、
　o Go言語のEntityを自動生成
　o Go言語のCDefを自動生成
が、できればと。
テンプレートを修正して検証しながらGo言語の第一歩を踏んでいきます。


# ========================================================================================
#                                                                              Environment
#                                                                              ===========
clone直後は、とりあえず ReplaceSchema を叩いてデータベースを作成します。

そして、manage.sh 23 を叩くと、src/main/go 配下に自動生成がされます。
テンプレートは、mydbflute/dbflute-1.x/templates/om/go 配下にあります。

※ソースコードの出力先は、basicInfoMap.dfprop にて調整できます。
